# 2212506060_Efe-Guler-Goruntu-Isleme-Dersi-Final-Projesi

	•	Gerçek zamanlı kamera kullanımı
	•	El ve parmak tespiti

  Kullanılan Teknolojiler
	•	Python 3.10
	•	OpenCV
	•	MediaPipe
	•	NumPy

Kurulum Ve Çalıştırma

Bilgisayarınızda Python 3.10 kurulu olmalıdır.
Python sürümünüzü kontrol etmek için terminale "python --version" komutunu yazabilirsiniz.

Proje dosyasını indirdikten sonra ilk işlem olarak terminalden sanal ortam oluşturmamız gerek bunun için terminale "python -m venv handenv" komutunu yazıyoruz ardından;

MacOs Kullanıcıları için sanal ortamı aktif etme:
source handenv/bin/activate

Windows Kullanıcıları için sanal ortamı aktif etme:
handenv\Scripts\activate

Sanal ortam aktif edildiğinde terminalin başında "handev" yazsını görmemiz gerek.

Daha sonrasında "python hand_draw.py" komutu ile projeyi çalıştırabiliriz.
